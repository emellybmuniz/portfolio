## Sobre o Projeto
O Projeto Portfólio foi inicialmente desenvolvido como requisito acadêmico para o curso de Engenharia de Software no Centro Universitário Internacional Uninter. O objetivo principal foi consolidar os conhecimentos básicos da Web ao entregar uma Single Page Application (SPA) moderna, fluida e totalmente funcional usando apenas Vanilla JS, HTML e CSS puro.

### Desenvolvimento Customizado

Ao invés de adotar templates ou frameworks como Bootstrap, a interface foi desenhada do zero. O projeto implementa um menu lateral dinâmico para desktop que se transforma em uma barra inferior de navegação no mobile (app-like), melhorando expressivamente o alcance em telas menores. Efeitos de fade-in e um botão de alternância de tema global são totalmente manipulados via JavaScript, evitando inflar o pacote do projeto.
