## Sobre o Projeto
Este projeto é uma simples landing page informativa simulando uma matéria de blog de tecnologia, construído como um forte exercício prático dos conceitos iniciais de desenvolvimento web e media queries.

### Responsividade na Prática

O foco técnico do projeto não estava na lógica de programação, mas no comportamento de adaptação fluida (Responsive Web Design). Em vez de definir larguras fixas, o conteúdo é inteiramente escalável e maleável, quebrando graciosamente de um fluxo de coluna única no mobile para distribuições maiores no desktop, focando em manter a usabilidade da leitura em níveis altíssimos.
