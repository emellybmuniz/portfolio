## Sobre o Projeto
O EB Cofrinho Java é uma aplicação baseada em linha de comando (CLI) projetada para consolidar os fundamentos de Programação Orientada a Objetos. O projeto simula o comportamento de um cofrinho virtual capaz de armazenar moedas de diversos países (Real, Dólar, Euro, etc.).

### Arquitetura de Classes

A estrutura foi planejada para demonstrar o forte encapsulamento do Java. A classe mãe `Moeda` dita os comportamentos abstratos, enquanto as classes específicas como `Dolar` e `Euro` aplicam as regras de polimorfismo para converter seus valores em Real com base em taxas de câmbio pré-definidas no momento do saque ou cálculo total.
