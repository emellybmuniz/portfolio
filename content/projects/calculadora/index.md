## Sobre o Projeto
O projeto da Calculadora Web foi desenvolvido para exercitar padrões de arquitetura de código front-end (como Factory Functions) ao invés de apenas criar um layout visual. 

### Lógica e Funcionalidade

A arquitetura JavaScript do projeto foca na segurança e encapsulamento. A lógica principal isola a captura de botões, o tratamento de eventos no DOM e o motor de cálculo. Há tratamento específico para entradas inválidas (ex: múltiplas operações ou divisão por zero) com o uso da propriedade `isFinite` do JavaScript, barrando inconsistências.

Além do clique, o projeto suporta navegação por teclado e otimizações de touch em dispositivos móveis, garantindo acessibilidade e usabilidade consistentes.
