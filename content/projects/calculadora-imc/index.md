## Sobre o Projeto
A Calculadora de Índice de Massa Corporal (IMC) foi projetada para prover um diagnóstico rápido baseado na fórmula internacional da OMS. O layout foi desenvolvido para focar diretamente na utilidade, contendo uma interface limpa baseada num card unificado.

### Implementação de Validação

O maior ganho no desenvolvimento deste projeto foi lidar com inconsistências de dados fornecidos pelo usuário. O script lida nativamente com as diferenças de pontuação decimal (aceitando tanto ponto quanto vírgula) e gera tratamento de erro claro se valores nulos ou inválidos forem submetidos, exibindo também as faixas de risco diretamente na tela após a ação primária.
