## Sobre o Projeto
Uma experiência gamificada totalmente feita no lado do navegador, criada para aprofundar a interação e disparo de eventos e alteração visual dinâmica em elementos de tela usando JavaScript nativo.

### Lógica Assíncrona e DOM

O jogo capta as intenções de clique dos botões e a resolve contra uma pseudo-inteligência programada pela classe `Math.random` do JS. As interações do usuário imediatamente reescrevem o painel de pontuação no DOM enquanto alteram classes CSS do placar, servindo como um sólido experimento prático de interações assíncronas de interface na web.
