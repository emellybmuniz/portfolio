## Sobre o Projeto
O Web Playground não é um produto finalizado, mas um canivete suíço. É uma coletânea mantida paralelamente de mini-experimentos com as tríades da web.

### Espaço Seguro para Erros

O projeto serve como uma incubadora. Funcionalidades como temporizadores isolados, requisições de teste para APIs públicas e ensaios com CSS SASS ou Hugo nasceram em isolamento aqui antes de migrar para os projetos oficiais. Ter este espaço dedicado ajuda a garantir a qualidade de entrega nos projetos finais simulando as lógicas previamente em um painel fechado e seguro.
