## Descrição rápida

Este repositório agrupa pequenos projetos, exercícios e recursos voltados ao desenvolvimento front-end. É pensado para experimentação: abrir arquivos estáticos, testar layouts, brincar com CSS e praticar JavaScript.

## Estrutura principal

| Pasta / Arquivo    | O que contém (resumo)                                                                 |
| :----------------- | :------------------------------------------------------------------------------------ |
| **challenges/**    | Desafios de código |
| **lab/**           | Exercícios e experimentos.             |
| **mini-projects/** | Pequenos projetos (clock, pomodoro, gerador de QR, site com SASS + Hugo).             |
| **README.md**      | Este arquivo — sumário e instruções rápidas.                                          |
| **index.html**      | Painel inicial para navegação facilitada                                         |

> Observação: abra a pasta desejada e acesse o arquivo `index.html` correspondente.

## Como visualizar localmente 

- Abra o `index.html` da raiz para entrar no painel inicial do projeto.
- Abrir o arquivo HTML diretamente no navegador (clique duplo em `index.html`).
- Servidor HTTP simples (recomendado para recursos locais/paths relativos):

```bash
# 1) Com Python 3 (porta 8000)
python -m http.server 8000

# 2) Com Node.js + http-server
npx http-server -c-1

# Abra http://localhost:8000/ e use o painel inicial para navegar
```
