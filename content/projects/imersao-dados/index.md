## Sobre o Projeto
Desenvolvido originalmente durante a Imersão Python da Alura, este projeto consiste em uma robusta ferramenta visual para analisar a distribuição salarial da área de Dados no Brasil e exterior. 

### Visualização e Exploração

O Dashboard foi projetado focado no usuário não-técnico. Em vez de ler planilhas massivas, o usuário interage visualmente usando múltiplos filtros (nível de senioridade, tipo de contrato e país), gerando recálculos matemáticos instantâneos nas métricas e plotando novos gráficos em tempo real. O uso estratégico da biblioteca Streamlit aliado ao Plotly transformou análises exploratórias acadêmicas em um painel interativo ágil, funcional em navegadores desktop e dispositivos móveis.
